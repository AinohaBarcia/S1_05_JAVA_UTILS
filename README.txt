Para poder ejecutar un proyecto desde la terminal tenemos que hacer lo 
siguiente:

EJERCICIO 1
1.- Navegar hasta la carpeta del proyecto.
2.- Una vez dentro de la carpeta java ejecutamos la siguiente sentencia:
	javac n1exerici1/Main.java
3.- Una vez ha compilado le indicamos el path relativo:
	java n1exerici1/Main ../../../resources

EJERCICIO 2
1.- Navegar hasta la carpeta de raiz del proyecto.
2.- Una vez dentro de la carpeta java ejecutamos la siguiente sentencia:
	javac n1exercici2/Main.java
3.- Una vez compilado le indicamos el siguiente path relativo
	java n1exercici2/Main ../../../resources

EJERCICIO 3
1.- Navegar hasta la raiz del proyecto.
2.- Una vez dentro de la carpeta java ejecutamos la siguiente sentencia:
	javac n1exercici3/Main.java
3.- Ahora le indicamos el path relativo
	java n1exercici3/Main ../../../resources

EJERCICIO 4
1.- Volvemos a la raiz del proyecto.
2.- Una vez dentro de la carpeta java ejecutamos la siguiente sentencia:
	javac n1exercici4/Main.java
3.- Ahora ejecutamos indicando el path relativo:
	java n1exercici4/Main ./n1exercici4/numbersList

EJERCICIO 5
1.- Vamos de nuevo a la raiz del proyecyo.
2.- Una vez dentro de la carpeta java ejecutamos la siguiente sentencia:
	javac n1exercici5/Main.java
3.- Indicamos el path relativo
	javac n1exercici5/Main
